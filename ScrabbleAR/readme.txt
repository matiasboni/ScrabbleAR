Proyecto: ScrabbleAR
    ScrabbleAR es un juego basado en el juego de mesa Scrabble, en el cual mediante la construcción 
    de palabras sobre un tablero se intenta ganar puntos.

Pre-Requisitos:

    Tener instalado:
            Nombre           |   Version utilizada:
        Pattern              |       2.6
        PySimpleGUI:         |       4.18.2
        Python:              |       3.6.8

    Sistema operativo:
                      -Windows(10)  
                      -Linux(Ubuntu 18.04.4)
        
        Las versiones especificadas anteriormente fueron las que se utilizaron.
        Es posible que funcione en otras versiones.

Ejecución de la aplicación.
    Una vez que cumpla con los requisitos mencionados anteriormente, para comenzar el juego debe ejecutar
    el modulo ScrabbleAR.py. Una vez inicializado el juego debera visualizar el menú con 4 botones:
    -JUGAR:le permite ingresar a la pantalla del tablero.Este varia dependiendo del nivel que por defecto es el nivel medio. 
           Para que se repartan las fichas y que el juego empiece debe presionar el boton INICIAR. 
    -CONFIGURACIÓN:le permite configurar el nivel, el puntaje y cantidad de cada conjunto de letras, el tiempo de la partida y de la jugada. 
    -RANKING:le permite visualizar el top 10 de puntajes de cada nivel.
    -SALIR:le permite salir del juego.
    
Construido con:
    -PySimpleGUI:se utilizo para las interfaces gráficas
    -Pattern:se utilzo para la verificación de las palabras.
