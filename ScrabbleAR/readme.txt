Proyecto: ScrabbleAR
    ScrabbleAR es un juego basado en el juego de mesa Scrabble, el cual consiste en armar 
    palabras sobre un tablero. Se juega contra la computadora y el que mayor puntos obtenga sera
    el ganador.

Pre-Requisitos:

    Tener instalado:
            Nombre           |   Versión utilizada
        Pattern:             |       2.6
        PySimpleGUI:         |       4.18.2
        Python:              |       3.6.8

    Sistema Operativo:
                      -Windows(10)  
                      -Linux(Ubuntu 18.04.4)
        
        Las versiones especificadas anteriormente fueron las que se utilizaron.
        Es posible que funcione en otras versiones.

Ejecución de la aplicación.
    Una vez que cumpla con los requisitos mencionados anteriormente, para comenzar el juego debe ejecutar
    el modulo ScrabbleAR.py, ya sea desde consola, editor de texto o un IDE. Una vez inicializado el juego deberá visualizar el menú con 4 botones:
    -JUGAR:le permite ingresar a la pantalla del tablero.Este varía dependiendo del nivel, que por defecto es el nivel medio. 
           Para que se repartan las fichas y que el juego empiece debe presionar el botón INICIAR. 
    -CONFIGURACIÓN:le permite configurar el nivel, el puntaje y cantidad de cada conjunto de letras, el tiempo de la partida y de la jugada. 
    -RANKING:le permite visualizar el top 10 de puntajes de cada nivel.
    -SALIR:le permite salir del juego.
    
Construido con:
    -PySimpleGUI:se utilizó para las interfaces gráficas
    -Pattern:se utilizó para la verificación de las palabras.
